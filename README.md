# BBC Basic Programs

A collection of some of the programs I wrote for the BBC Micro Model B, in the late 80's and early 90's.

## Run them in your browser

**https://tjrobinson.github.io/BBCBasicPrograms/** — every program in this repository, running on an
emulated BBC Micro ([jsbeeb](https://github.com/mattgodbolt/jsbeeb)) directly in the browser.
The site is entirely self-contained: a vendored build of jsbeeb lives in [jsbeeb/](jsbeeb/) and is
served by GitHub Pages alongside the programs, so it has no external dependencies.

To run it locally: `python3 -m http.server` from the repository root, then open http://localhost:8000/.

## Tools

* [JSBeeb](https://bbc.godbolt.org/)
* [BBC BASIC for SDL 2.0](http://www.bbcbasic.co.uk/bbcsdl/index.html)
